<JAMASTパッケージ概要>

[org.mklab.mikity]
	┗[action]SWTで実行する内容とまとめたパッケージ
		┗[file]ファイルの読み取りや保存を行うためのパッケージ
		┗[toolbar]ツールバーを使用するためのパッケージ
	┗[gui]GUI関係のパッケージ
		┗[collision]プリミティブの重複防止を行うパッケージ。(ただし、現在未実装)
		┗[connector]プリミティブ同士を接続するコネクタを操作するパッケージ。
		┗[dialog]プリミティブやプリゴン、グループの追加や編集ダイアログのパッケージ。
		┗[dnd]ドラッグアンドドロップを可能にするためのパッケージ
	┗[internal]プリミティブの生成や環境設定といった内在的なパッケージ(現在未実装？)
	┗[picker]アニメーション再生時に現時刻と再生する行列データの時間で一番近いもののデータを取るためのパッケージ
	┗[java3D]java3Dで描画するパッケージ
	┗[jogl]joglで描画を行うパッケージ
		┗[models]モデルを表すパッケージ
	┗[resource]GUIのアイコンのリソースに関するパッケージ
	┗[task]アニメーションを実行するためのパッケージ
	┗[util]色やメッセージといった有用性に関するパッケージ
	┗[xml]モデリングデータをまとめ、データの読み取りや保存を行うパッケージ。
		┗[blender]Blenderから出力したCOLLADAデータを読み込むパッケージ。
		┗[config]JAMASTで作成したデータのモデリング部分に関するデータをまとめるパッケージ
		┗[model]JAMASTで作成したデータのモデリング環境に関するデータをまとめるパッケージ
